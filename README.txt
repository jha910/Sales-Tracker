Chrome Extension POC Setup Instructions

1. Ensure Node.js is installed: https://nodejs.org/
2. Unzip the folder.
3. Double-click install_and_start.bat to install dependencies and set up auto-start.
4. Load the extension in Chrome:
   - Go to chrome://extensions/
   - Enable Developer Mode
   - Click "Load unpacked"
   - Select the unzipped folder
5. Replace MongoDB credentials in the .env file with your actual values.
